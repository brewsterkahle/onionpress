### 3. HOW IT WORKS PAGE

**Title**: From Download to Live Site in 5 Minutes

**Steps** (with screenshots):

1. **Download Onion.Press**
   - One file, 80MB
   - Compatible with Mac (Intel & Apple Silicon)
   - macOS 13 or later

2. **Install & Launch**
   - Drag to Applications folder
   - Open the app
   - It lives in your menubar

3. **First Launch Magic**
   - Onion.Press downloads WordPress
   - Sets up Tor hidden service
   - Generates your .onion address
   - Takes ~2 minutes

4. **Your Site is Live!**
   - Click "View Site" to see it
   - Access WordPress admin
   - Start creating content

5. **Share Your Address**
   - Your .onion URL works anywhere
   - Share with Tor Browser users
   - Accessible worldwide

**Under the Hood** (for curious folks):
- Uses Colima (lightweight containers)
- WordPress 6.7 + MariaDB database
- Tor hidden service (v3 onions)
- All runs locally on your Mac
- No cloud, no servers, no data leaves your machine

**Mission**:

Remember the early web? When anyone could make a homepage? When the internet was weird and wonderful and *yours*?

## Implementation Notes

**WordPress Setup**:
1. Install Twenty Twenty-Four theme
2. Create pages as above
3. Add custom CSS for retro styling
4. Install plugins:
   - Contact Form 7 (for contact page)
   - Simple CSS (for custom styling)
   - Internet Archive Plugin (if we make one)

**Custom CSS** (retro style):
- Bold borders on everything
- Bright link colors
- Pixel fonts (Press Start 2P from Google Fonts)
- Table-style layouts (but using CSS Grid for mobile)
- Animated underlines on links
- GIF backgrounds (subtle)

**Performance**:
- Optimize images (even GIFs)
- Lazy load images
- Keep it fast on Tor (bandwidth matters)

**Accessibility**:
- Despite retro style, maintain proper heading structure
- Alt text on all images
- Keyboard navigation
- Color contrast that works
- Screen reader friendly
